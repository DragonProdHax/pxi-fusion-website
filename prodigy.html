Prodigy Installation Tutorial

Follow these steps to install PXI Fusion:

Step 1: Download the PXI Fusion extension file.

Step 2: Open the Chrome Extensions Page by clicking the three dots in the top-right corner, selecting "More Tools," and then "Extensions." Or, type chrome://extensions/ in the address bar.

Step 3: Enable Developer Mode on the Extensions page by toggling the switch in the upper-right corner.

Step 4: Click "Load unpacked," then choose the folder containing the unzipped PXI Fusion files (not the ZIP file itself).

For more details, you can also watch the video tutorial.
