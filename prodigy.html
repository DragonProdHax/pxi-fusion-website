Prodigy Cheat Installation Instructions
Follow these steps to install the PXI Fusion Extension:

Step 1: Install The PXI Fusion Extension

Download the PXI Fusion ZIP file from the official repository: PXI Fusion Extension ZIP

Step 2: Open the Chrome Extensions Page

Open Google Chrome. Click on the three dots in the top-right corner of your browser. Go to More Tools > Extensions. Alternatively, type chrome://extensions/ in the address bar and press Enter.

Step 3: Enable Developer Mode

On the Extensions page, toggle the Developer mode switch in the upper-right corner.

Step 4: Upload PXI Fusion

Click the "Load unpacked" button that appears after enabling Developer mode. Select the folder containing the unzipped PXI Fusion files.
